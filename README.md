## Installation

RQKit is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'RQKit', '0.1.4'
pod 'RQKit', :git => 'https://gitee.com/yin_gu/RQKit.git', :tag => '0.1.4'
```
